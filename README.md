# Almohandes-app
Makteb ALMohandes APPS

متجر تطبيقات ثابت (Static Site) لعرض وتحميل تطبيقات أندرويد / TV.

## البنية
- `index.html` — الصفحة الرئيسية
- `style.css` — التصميم
- `data.json` — قائمة التطبيقات (اسم، أيقونة، رابط تحميل...)
- `appLoader.js` — يبني بطاقات التطبيقات من `data.json`
- `script.js` — البحث ونسخ الروابط
- `img/` — أيقونات التطبيقات

## إضافة تطبيق جديد
أضف عنصراً جديداً في `data.json` وارفع أيقونته في مجلد `img/`.
